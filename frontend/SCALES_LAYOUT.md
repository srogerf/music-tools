# Scale Layout Rules

The canonical scale-layout design doc now lives at `docs/SCALE_LAYOUTS.md`.

This frontend note should stay lightweight and only record UI-local details:

- the CAGED selector order is defined in `frontend/app/scales/scales_layout.js`
- the scales page control state lives in `frontend/app/scales/scales_page.js`

Update `docs/SCALE_LAYOUTS.md` when the layout model or algorithm changes.
